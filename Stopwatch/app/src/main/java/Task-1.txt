1. Приложение-секундомер.
2. Простой отсчет секунд 1-2-3.. и т.д.
3. Цифры максимально крупные и отображаются по центру экрана. При этом стоит учитывать текущее
   положение устройства - при ландшафтной ориентации цифры чуть меньше, при портретной - чуть
   больше.
4. При перевороте экрана секундомер не должен останавливать или сбрасываться.
5. Предусмотреть кнопку "reset" в ActionBar для сброса секундомера.
6. Архитектура должна быть продумана, поскольку, как и написал в начале, в дальнейшем будем
   добавлять другие фичи в этот же проект.
7. Минимальная версия API - 17 (Android 4.2)
