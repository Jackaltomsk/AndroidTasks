Следующее задание будет связано с работой с сетью. Основные компоненты, построение интерфейса, БД и
прочие мы изучили (основы, само собой).
Поскольку мы сходу не придумали другое API, то предлагаю воспользоваться сервисом oboobs.ru :) У
него есть полностью открытое API. Описание скудновато, конечно: http://api.oboobs.ru/

Что надо дополнительно к нему знать:
1. Базовый URL для запросов - api.oboobs.ru
2. Базовый URL для загрузки картинок - media.oboobs.ru

Необходимо добавить в таймер возможность использования фона с этого сайта :) Таким образом:
1. В настройках приложения добавить пункт "Фоны oboobs"
2. Открывать новую Activity, в которой должен быть RecyclerView (вертикальный скроллинг, 2
    изображения в ряд)
3. Загрузка постраничная (дойдя до конца страницы, подгружать следующую)
4. У загрузки должна быть индикация - ProgressBar внизу списка (при первой загрузке, когда список
    еще пуст он будет вверху)
5. По клику на изображение оно должно открываться в полном экране (лучше здесь реализовать отдельный
    фрагмент, активити излишне уже) с возможностью pinch-to-zoom.
6. В полном экране должна быть кнопка "Установить изображение", по нажатию на которую фон таймера
    должен меняться на выбранную картинку.
7. При перевороте экрана число столбцов должно меняться с 2х (в портретной) до 4х (в ландшафтной).

Небольшой совет: для работы с API можно использовать гугловскую библиотеку Volley. В ней же есть
класс ImageLoader, который реализует асинхронную загрузку картинок и их кэширование.
