1. Экран настроек (переход из меню).
2. В настройках добавить опцию выбора цвета фона. Цветов не меньше 5.
3. После выбора цвета, экран настройки должен закрываться, а фон таймера меняться плавно с
   анимацией. Без резких смен цвета.
4. После запуска таймера он не должен останавливаться без команды юзера (нажатие на "остановить"
   или выход из приложения). В шторке уведомлений надо добавить неудаляемую нотификацию с текущим
   временем. По нажатию на нее - переход в приложение.
